Subject : New function! Device information!

As the subject already says, just more specific information about the system/OS e.g.

The name of the operating system (e.g. Windows 10, Ubuntu, macOS, etc.).
Version of the system
Version and build number of the operating system (e.g. Windows 10 version 22H2, Ubuntu 20.04 LTS).
License
The license type under which the operating system is distributed (e.g. open source, proprietary, freeware, commercial).
License number / key
For commercial operating systems (such as Windows or macOS), a license number or product key.
System architecture
Architecture of the system (e.g. 32-bit, 64-bit).
Processor (CPU)
Details of the CPU used (e.g. Intel Core i7, AMD Ryzen 5, clock frequency).
Memory (RAM)
Amount of RAM installed (e.g. 8 GB, 16 GB).
Hard disk storage / SSD
Capacity and type of storage (e.g. 500 GB HDD, 1 TB SSD).
Graphics card
Information on the installed graphics card (e.g. NVIDIA GeForce GTX 1660, AMD Radeon RX 580).
Network cards / WLAN
Information on the network card and WLAN function (e.g. 1 Gbit Ethernet, 802.11ac WLAN).
Drivers
Information on installed drivers and their versions.
Installed software
A list of the installed software and applications running on the operating system.
User accounts
Information on local users or domain users (e.g. administrator rights, group memberships).
Security functions
Information about the update status (e.g. automatically installed updates, last system update).
Operating system status
System status such as system integrity, activation status (whether the operating system is activated) and hardware detection.
Language settings
The current system language and regional settings.
Energy settings
Information on the energy options (e.g. energy-saving mode, screen brightness).
Virtualization functions
Whether virtualization functions such as Hyper-V (Windows) or KVM (Linux) are activated.
Connected devices
